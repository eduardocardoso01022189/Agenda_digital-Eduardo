🚀 Minha Agenda Digital PITON

Organize seus estudos de forma inteligente

Este aplicativo foi desenvolvido utilizando HTML, CSS e JavaScript. O objetivo é criar uma ferramenta simples para cadastrar, acompanhar e organizar tarefas e matérias escolares.

A plataforma permite registrar atividades, acompanhar o progresso dos estudos e manter uma rotina organizada durante todo o ano letivo.

O sistema foi construído seguindo o modelo Dashboard, permitindo uma navegação rápida e intuitiva entre as funcionalidades.

Através da área de tarefas o estudante pode adicionar atividades, visualizar a quantidade total cadastrada e marcar conteúdos já concluídos.

  • ✔ Cadastro de tarefas
  • ✔ Interface Dashboard
  • ✔ Tema claro e escuro
  • ✔ Contador automático
  • ✔ Marcação de tarefas concluídas
  • ✔ Organização de matérias
  • ✔ Navegação por menu lateral
